## 测试建议

### 测试场景1：全新安装
1. 卸载现有 OneClip（如果已安装）
2. 运行 `./install.sh install`
3. 验证安装成功
4. 检查应用是否正常运行

### 测试场景2：版本升级
1. 确认当前版本
2. 运行 `./install.sh upgrade`
3. 验证升级到最新版本
4. 检查数据是否保留

### 测试场景3：问题修复
1. 运行 `./install.sh diagnose`
2. 查看诊断结果
3. 选择自动修复
4. 验证问题是否解决

### 测试场景4：数据备份恢复
1. 运行 `./install.sh backup`
2. 验证备份文件创建
3. （可选）删除部分数据
4. 运行 `./install.sh restore`
5. 验证数据恢复成功

### 测试场景5：完全卸载
1. 运行 `./install.sh uninstall`
2. 选择"完全卸载"
3. 验证应用和数据已删除
4. 验证 Tap 已移除
